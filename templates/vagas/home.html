Como usar o Garimpo

Guia rápido para analisar a compatibilidade entre currículo e vagas de tecnologia.

1. Crie uma conta ou entre no sistema

O login é opcional, mas recomendado. Com uma conta, seus currículos e resultados ficam vinculados ao seu usuário.

2. Envie seu currículo

Acesse Enviar Currículo, selecione um arquivo PDF e informe um nome para identificação, como “Currículo Principal”.

3. Atualize as vagas

Clique em Atualizar Vagas para buscar oportunidades reais em uma fonte externa e salvar os dados no banco.

4. Calcule o match

Em Ver Vagas, selecione um currículo e calcule a compatibilidade técnica com cada vaga.

5. Veja os detalhes

Após calcular o match, o sistema mostra as competências alinhadas, as competências faltantes e a explicação do cálculo utilizado.

6. Acompanhe pelo dashboard

No Dashboard, selecione uma vaga e visualize o percentual de compatibilidade com o currículo escolhido.

Comece a análise

Para obter o melhor resultado, siga a ordem: envie o currículo, atualize as vagas, calcule o match e consulte o dashboard de compatibilidade.
